== Description ==

PHP Render is a WordPress plugin that allows you to execute and edit PHP code from files using shortcodes. It provides a shortcode to render PHP code from specified files and a PHP code editor within the WordPress admin area.

== Features ==

- Execute PHP code from files using shortcode.
- Edit, create, delete, and manage PHP files within the WordPress admin area.
- Import and export PHP files using JSON format.
- Documentation page with usage instructions.

== Installation ==

1. Upload the `php-render` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[render-php]` shortcode to execute PHP code from files.
4. Use the `[php_editor]` shortcode to access the PHP code editor.
